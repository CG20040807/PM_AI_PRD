# 🚀 AI Product Manager Agent

一个 **AI 产品经理助手**，输入产品名称即可自动生成完整产品分析，包括 **PRD、用户画像、竞品分析和商业模式**。

该工具可以帮助产品经理快速完成产品调研和PRD初稿，提高产品设计效率。

---

## 🌟 项目演示

输入产品名称，例如：

```
抖音
```

系统将自动生成：

* 📊 产品背景分析
* 👥 用户画像分析
* ⚙️ 核心功能拆解
* 📈 竞品分析
* 💰 商业模式分析
* 📝 PRD结构文档

并支持 **一键下载PRD文档**。

---

## ✨ 功能特性

* 🚀 **AI PRD生成**
  自动生成完整产品需求文档（PRD）

* 👥 **用户画像分析**
  AI分析目标用户群体

* 📈 **竞品分析**
  自动生成主要竞品对比

* 💰 **商业模式分析**
  输出产品商业模式与盈利方式

* 📥 **文档下载**
  一键下载完整产品分析文档

---

## 🖥️ 产品界面

Web界面提供：

* 产品名称输入
* AI分析结果展示
* PRD下载功能

运行后访问：

```
http://localhost:8501
```

## 📦 安装运行

### 1️⃣ 克隆项目

```
git clone https://github.com/yourname/ai-product-manager-agent.git
```

进入项目目录：

```
cd ai-product-manager-agent
```

---

### 2️⃣ 安装依赖

```
pip install -r requirements.txt
```

---

### 3️⃣ 启动项目

```
streamlit run app.py
```

浏览器打开：

```
http://localhost:8501
```
